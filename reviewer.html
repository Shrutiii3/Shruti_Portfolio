Code Reviewer

AI Powered Code Assistant

  • Developed an AI-powered web application that not only reviews code for syntax and quality but also analyzes the problem statement to detect logical errors and explain failing test cases.

  • Integrated a real-time code editor to support syntax highlighting for 3+ languages and improve the overall user experience.
  • Architected a dedicated REST API endpoint using Node.js and Express.js, using effective prompt engineering to generate AI feedback across 4+ categories including performance, potential bugs and best practices.
